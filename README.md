# 幸福市场 · Happiness Market

> 从剥削体系到幸福生态：一个社会重构的蓝图

**幸福市场**是一个沉浸式叙事网页，通过三个不同阶层人物的生命故事，构想一种不以货币为核心驱动力，而是以全民多维幸福感为导向的社会形态。

## 核心理念

旧体系的根基在于将"幸福"私有化、等级化。新市场从根本上改变游戏规则——**将幸福从被少数人垄断的"名词"，转变为全民参与、动态流动的"动词"**。

### 五大重构步骤

1. **定义权起义** — 砸碎"幸福"的单一标准，确立以身体活力、精神宁静、社会联结、生命意义、自主性、生态安全为维度的多维幸福宪法

2. **解除生存胁迫** — 通过无条件基本收入（UBI）和时间银行，赋予每个人说"不"的权利，让照护、陪伴等"爱的劳动"获得硬通货般的认可

3. **经济民主化** — 推行企业民主所有制，利润由全体成员民主决定用途，建立"贡献可视化"系统让每份劳动的意义直接可见

4. **光荣的退出** — 为旧既得利益者设计转化通道：物质奢靡升级为创造者的心流，控制欲升级为深度联结，财富焦虑升级为永恒尊荣

5. **永续流动** — 关键岗位抽签轮流制 + 全民幸福体验"轮作"，防止任何群体垄断高幸福感生态位

## 三位人物 · 四种人生阶段

| 人物 | 阶层 | 核心叙事 |
|------|------|----------|
| **林禾** | 底层 | 从被迫接单的零工青年，成长为社区最可靠的照护与修复组织者 |
| **周岚** | 中层 | 从焦虑竞争转向协作生活，在互助住房与社区共治中找到归属 |
| **顾承** | 上层 | 从占有转向共建与荣耀退出，通过社会遗产信托获得真正的尊严 |

每个人物展开为四个生命阶段，配有专属插画，展示每次参与如何改变个人命运和整个共同体。
